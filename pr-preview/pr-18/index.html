Foreman¶

A Python harness that acts as an always-on AI co-maintainer for OSS repositories.

What Next¶

To get started, see Installation, then follow the Quick Start guide. For a deeper understanding of how Foreman works, read How Foreman Works.

Once you're up and running, check out the Reference for technical details on configuration, the CLI, and the agent protocol. Developers extending Foreman should read the Developer Guide.
